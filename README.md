# Injection de dépendances

## Objectif
Implémenter l'injection de dépendances en Java.

## Étapes
- Création des interfaces IDao et IMetier
- Implémentations
- Injection statique
- Injection dynamique
- Injection avec Spring (XML & Annotations)

## Résultat
Le programme calcule une valeur basée sur les données fournies par Dao.
